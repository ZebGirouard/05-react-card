# 05 React Card

## Goal
Rebuild the personal card as React components so you can see how props and state shape UI.

## What You Are Practicing
- What a React component returns
- What `props` are for
- What state is for
- How a click handler can update the UI

## Start Here
1. Run `npm install`.
2. Run `npm run dev`.
3. Open `src/App.jsx` and find the `isAvailable` state near the bottom of the file.
4. Look at the status button that already reads from that state.
5. Make the button react to a click so the label changes in the browser.

## Stretch Goals
- Add a second card from a new data object.
- Extract the status button into its own component.
- Let the accent color change based on state.
