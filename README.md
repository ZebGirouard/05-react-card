# 05 React Card

## Goal
Rebuild the personal card as React components so you can see how props and state shape UI.

## Start Here
1. Run `npm install`.
2. Run `npm run dev`.
3. Open the local URL and make the status button actually change the interface.

## Stretch Goals
- Add a second card from a new data object.
- Extract the status button into its own component.
- Let the accent color change based on state.
